HTML:5


HTML não é uma linguagem de programação; é uma linguagem de marcação, usada para definir a estrutura do seu conteúdo. HTML consiste de uma série de elementos, que você usa para delimitar ou agrupar diferentes partes do conteúdo para que ele apareça ou atue de determinada maneira.

Geralmente as principais partes de um elemento html contém:

  • Tag de abertura.
  • Tag de fechamento (Nem todos elementos possuem.)
  • Conteúdo
  • Os elementos que compõe a tag.

Formatação de textos


Existem algumas tags com a semântica necessária para englobar textos, palavras e outros termos no HTML. Quando necessário devemos utilizar essas tags para dar um melhor significado no que estamos escrevendo.

Segue abaixo um pequena lista com suas respectivas funções, lembre-se que todas essas tags necessitam de abertura e fechamento para fazerem sentido.

  • <b> - Texto em Negrito
  • <strong> - Texto importante
  • <i> - Texto em itálico
  • <em> - Texto com ênfase
  • <mark> - Texto marcado
  • <small> - Texto pequeno
  • <del> - Texto deletado
  • <ins> - Texto inserido
  • <sub> - Texto subscrito
  • <sup> - Texto sobrescrito

Esta tag, não serve para formatar textos, mas é útil para exibir divisões <hr> (horizontal row) ela cria uma linha horizontal na página.

listas


As listas são representações de textos em forma ordenada.

Temos listas ordenadas, listas sem ordem, listas de descrição. <ol></ol>, <ul></ul>, <dl></dl>, segue exemplo abaixo.

A lista ordenada e não ordena utilizam dos UL e LI para definir seus itens. A lista de descrição utiliza a marcação DL para definir o tipo de lista e DT para definir o termo e DD para descrever o termo.

imagens


Para inserir uma imagem no corpo do nosso site é necessário o uso da tag <img>. Esta tag não necessita de fechamento e pede um atributo chamado alt="" que ajuda os leitores de tela a interpretarem o que está sendo exibido. Geralmente descrevemos o que há na imagem pois é importante para que deficientes visuais possam ter uma maior acessibilidade. Atenção para a palavra acessibilidade. Conforme formos avançando nesta área, será de suma importância investir em acessibilidade.

O atributo "alt" não aparece no corpo do site. Apenas nos leitores de tela.

Podemos inserir imagens de dentro do nosso próprio site ou de links externos. Eu recomendo sempre ter as imagens em seu banco de dados pois imagens de links externos podem ser removidas e sua exibição será prejudicada.


exemplo de como usar a tag image quando ela esta salva no seu diretório.


<img src="imagens/nome_da_imagem.formato" alt="descrição curta e objetiva da imagem">

Para inserir uma imagem direto da internet, substitua o conteúdo de dentro das aspas do src="" pelo link da imagem que desejar. exemplo


<img src="https://i.ytimg.com/vi/QjZv2ukqb_g/hqdefault.jpg" alt="terra vista do espaço">

Dentro da tag img, podemos colocar outros atributos, como width, height, para definir o tamanho da imagem em questão.

links


Links ou hiperligações são formas de encontrar ou ir para outras informações. Para inserir um link na página devemos inserir a tag <a> </a> com isso criamos uma âncora, porém, para navegar por links devemos inserir o href="" que será o responsável por direcionar nossa requisição.

dentro da tag link podemos encontrar alguns outros atributos importantes como o target="_atributo , onde cada um terá sua função especificada abaixo.


  • _self - Default. Abre o documento na mesma janela aonde foi clicado
  • _blank - Abre o link em outra janela
  • _parent - Abre um documento em uma aba próxima
  • _top - Abre o documento no corpo da janela

URLs absolutos vs. URLs relativos

Um link local (um link para uma página no mesmo site) é especificado com um URL relativo (sem a parte "https: // www"): Links


Use uma imagem como link


Para usar uma imagem como link, basta colocar a tag <img> dentro da tag <a>.


Link para um endereço de e-mail


Use mailto: dentro do atributo href para criar um link que abre o programa de e-mail do usuário (para permitir que ele envie um novo e-mail):
Exemplo
<a> href="mailto:someone@example.com"> Enviar e-mail </a>
Resultado: Enviar e-mail
O estilo da fonte, a cor, tamanho e tudo mais, pode ser formatado pelo CSS.

tabelas


Definindo uma tabela

A tag <table> define uma tabela HTML. Cada linha da tabela é definida com a tag <tr>. O cabeçalho da tabela é definido com tag <th>, cada célula da tabela é definida pela tag <td>.


Por predefinição o texto dentro de <th>, é negrito e centralizado.

Por predefinição o texto dentro de <td> são alinhados à esquerda.


Exemplos:



                            <table style="width: 100%">
                                <tr>
                                    <th>Nome:</th>
                                    <th>Sobrenome</th>
                                    <th>Idade</th>
                                </tr>
                                <tr>
                                    <td>Marcos</td>
                                    <td>Paulo</td>
                                    <td>50</td>
                                </tr>
                                <tr>
                                    <td>Evellin</td>
                                    <td>Smith</td>
                                    <td>36</td>
                                </tr>
                            </table>

Adicionando CELL PADDING


Cell padding é especificamente adicionar um espaço entre o conteúdo da célula e suas bordas.

Para adicionar um padding a uma th ou td devemos inserir o atributo padding: ; e um valor para que o espaçamento exista.

Exemplo:

                th, td {
                    padding: 15px;
                  }

Alinhamento à esquerda e Cabeçalhos

Por padrão os elementos são alinhados à direita e os cabeçalhos tem a fonte em negrito. Para alinhar os itens do cabeçalho ou dos dados gerais à esquerda, th, td utilize no csso seguinte 

        th, td {
            text-align: left;
        }

Outras formatações e novas adições


Podemos formatar o espaçamentos das bordas caso elas não estejam formatadas para com collapse. Utilizaremos o table { border-spacing: 5px; } isto adicionará um espaço de 5 pixels entre as bordas dos elementos.
